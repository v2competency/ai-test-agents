# Automation Generator Agent - Prompt Templates

> Ready-to-use prompts for invoking the Automation Generator Agent

---

## Quick Start Prompt

```
I have manual test cases ready for automation generation.

Test Case File: manual-tests/{AppName}_Manual_Test_Cases.txt
Base URL: {YOUR_APP_URL}

Please:
1. Parse the manual test cases
2. Show me what will be generated (pages, tests, data files)
3. After my confirmation, generate the complete Playwright automation framework
4. Save to playwright-projects/{AppName}/
```

```

I have manual test cases generated by the Screenshot Analyzer Agent for {APPLICATION_NAME}.

1. Parse all test cases from the file
2. Generate Page Object Model classes for each identified page
3. Create data-driven test specifications
4. Include JSON test data files organized by test type
5. Add self-healing locator utilities
6. Generate all configuration files

OUTPUT:
- Complete Playwright project in: playwright-projects/{AppName}/
- Project should be ready to run with: npm install && npm test

PREFERENCES:
- Browser: Chromium only
- TypeScript strict mode
- Include self-healing: Yes
- Generate README: Yes
```

## Tips for Best Results

1. **Use clean manual test files** - Ensure the input file follows the expected format
2. **Provide accurate Base URL** - This is used in config and navigation
3. **Review generation plan** - Check the plan before confirming generation
4. **Test incrementally** - Generate and test one module at a time for large projects
5. **Customize locators** - After generation, update locators based on actual DOM
6. **Add real test data** - Replace placeholder values with actual test credentials
7. **Run in headed mode first** - Use `npm run test:headed` to verify visually
