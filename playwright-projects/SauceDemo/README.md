## Quick Start

### Prerequisites

- Node.js >= 18.0.0
- npm or yarn

### Installation

```bash
# Clone or navigate to the project
cd playwright-projects/SauceDemo

# Install dependencies
npm install

# Install Playwright browsers
npx playwright install
```

### Running Tests

```bash
# Run all tests
npm test

# Run tests with UI
npm run test:ui

# Run tests in headed mode (see browser)
npm run test:headed

# Run tests with debugging
npm run test:debug

# Run specific test file
npm run test:auth
npm run test:inventory
npm run test:cart
npm run test:checkout
npm run test:e2e

# Run on specific browser
npm run test:chromium
npm run test:firefox
npm run test:webkit
npm run test:mobile
```

### Viewing Reports

```bash
# Open HTML report
npm run report
```

## Configuration

### Environment Variables

Copy `.env.example` to `.env` and configure:

```bash
BASE_URL=https://www.saucedemo.com
CI=false
HEADLESS=true
```

### Playwright Config

Key settings in `playwright.config.ts`:

- **Parallel execution**: Enabled by default
- **Retries**: 2 in CI, 0 locally
- **Timeout**: 60 seconds per test
- **Browsers**: Chromium, Firefox, WebKit, Mobile
- **Artifacts**: Screenshots on failure, videos on retry

## CI/CD Integration

### GitHub Actions Example

```yaml
name: Playwright Tests

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: 18
      - name: Install dependencies
        run: npm ci
      - name: Install Playwright Browsers
        run: npx playwright install --with-deps
      - name: Run tests
        run: npm test
        env:
          CI: true
      - uses: actions/upload-artifact@v4
        if: always()
        with:
          name: playwright-report
          path: reports/
```

## Extending the Framework

### Adding a New Page Object

```typescript
// pages/NewPage.ts
import { Page, Locator } from '@playwright/test';
import { BasePage } from './BasePage';

export class NewPage extends BasePage {
  readonly pageUrl: string = '/new-page';
  readonly someElement: Locator;

  constructor(page: Page) {
    super(page);
    this.someElement = page.locator('[data-test="element"]');
  }

  async someAction(): Promise<void> {
    await this.someElement.click();
  }
}
```

### Adding Test Data

Add scenarios to the appropriate JSON file in `/data`:

```json
{
  "newScenario": {
    "testId": "TC_NEW_001",
    "description": "New test scenario",
    "input": "value",
    "expectedResult": "success"
  }
}
```

## Troubleshooting

### Common Issues

1. **Tests timeout on login**: Increase timeout for `performance_glitch_user`
2. **Stale element errors**: Use proper wait strategies in page objects
3. **Session issues**: Clear storage between tests in beforeEach

### Debug Mode

```bash
# Run with Playwright Inspector
PWDEBUG=1 npm test

# Run specific test in debug mode
npx playwright test -g "TC_AUTH_001" --debug
```
